Labubu Classifier

Live Application: Labubu Classifier

A web-based image classification application built with Flask and PyTorch to distinguish authentic Labubu figures from visually similar counterfeit variants using a fine-tuned ResNet18 model.

Project Overview

This project was built to explore a practical computer vision problem: Can a lightweight deployed model help distinguish authentic collectibles from counterfeit lookalikes?

In this case, the focus was on identifying authentic Labubu figures versus fake variants often referred to by collectors as “Lafufu.” The result is a complete end-to-end machine learning project that covers data collection, augmentation, model training, web deployment, and real-time inference through an accessible user interface.

What Was Built

  • Binary image classification model using ResNet18 in PyTorch
  • Flask web application for image upload and prediction
  • Confidence score output for model decisions
  • Deployment pipeline using Render
  • Supporting scripts for dataset collection, augmentation, evaluation, and visualization

System Pipeline

Image upload → Flask app → preprocessing
            → ResNet18 inference → softmax probabilities
            → predicted class + confidence score

Key Engineering Decisions

  • Used ResNet18 as a lightweight and efficient architecture for binary image classification
  • Built the application in Flask for simple deployment and minimal overhead
  • Used CPU-friendly inference to support low-cost hosted deployment on Render
  • Handled uploaded image preprocessing consistently for real-time predictions
  • Structured the project with separate scripts for training, evaluation, augmentation, and visualization

What I Learned

  • How to turn a trained computer vision model into a usable web application
  • How to deploy PyTorch inference workflows in a lightweight production setting
  • How dataset quality and augmentation affect binary classification performance
  • How to design a simple interface that makes ML output understandable to users
  • How to debug deployment and image preprocessing issues in a real hosted application

Why This Matters

This project demonstrates more than model training alone. It shows the ability to build and deploy an end-to-end machine learning system that connects data, inference, web development, and user interaction.

It also reflects a practical application of computer vision to real-world authenticity and classification problems.
